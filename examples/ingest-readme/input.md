# Example onboarding handbook

A small markdown document used by the `ingest-readme` example.

## Setup

Install the toolchain and request access to staging.

### Toolchain

You'll need Node 18 or newer, Bun, and Git.

### Access

Request access to the staging cluster and the observability dashboard.

## Working norms

We default to async communication. Meetings are for decisions, not status.

### Code review

All code lands behind a pull request with at least one approval.
